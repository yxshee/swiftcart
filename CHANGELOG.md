# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial release of ShopNow SwiftUI Ecommerce App
- Product listing with grid/list view toggle
- Advanced filtering by category, price range, and availability
- Multi-sort options (newest, price, rating, popularity)
- Infinite scroll pagination
- Real-time product search
- Featured deals section
- Shopping cart with quantity controls
- Free shipping progress tracker
- Multi-step checkout flow (Shipping → Payment → Review)
- Multiple payment methods support
- Order confirmation with animations
- User profile with order history
- Dark mode support
- Custom gradient theme
- Smooth animations and transitions
- Empty state views
- Loading indicators with shimmer effects
- Error handling UI
- Cross-platform compatibility (iOS/macOS build support)
- Complete API documentation
- Sample product data for demonstration

### Technical
- MVVM architecture
- @EnvironmentObject for shared state management
- Async/await networking layer
- SwiftUI 3.0+ features
- iOS 18.0+ deployment target
- Zero external dependencies
- XcodeGen project generation
- Comprehensive documentation

### Fixed
- UIKit import compatibility with conditional compilation
- Cross-platform color support
- Haptic feedback iOS-only implementation
